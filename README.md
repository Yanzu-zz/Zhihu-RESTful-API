> 本项目是 RESTful API风格模仿知乎的一个项目

## 技术栈
  * NodeJS
  * Koa, Koa-router, Koa-parameter, ...
  * MongoDB

### 项目基础目录及文件功能
```
  ├─app
  │  ├─config       公共配置
  │  ├─controllers  控制器
  │  ├─models       数据控 schema 模型
  │  ├─public       公共资源
  │  │  └─uploads
  │  │─routes       路由
  │  └─index.js     主控制文件
```

## 安装和运行
```
  npm install
  npm run dev
```
